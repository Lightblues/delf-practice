DELF Practice App - Installation Guide
======================================

由于此应用未签名，macOS 会阻止直接运行。必须通过命令行安装。

方法一：使用安装脚本（推荐）
--------------------------
1. 打开终端 (Terminal)
2. 进入 DMG 挂载目录: cd /Volumes/DELF*
3. 运行安装脚本: bash install.sh
4. 按提示操作即可

方法二：手动安装
--------------
1. 打开终端 (Terminal)
2. 进入 DMG 挂载目录: cd /Volumes/DELF*
3. 运行: xattr -cr "DELF Practice.app"
4. 将 app 拖动到 Applications 文件夹


Installation (English)
======================

This app is unsigned, so macOS will block it. You must install via command line.

Method 1: Use Install Script (Recommended)
------------------------------------------
1. Open Terminal
2. cd /Volumes/DELF*
3. Run: bash install.sh
4. Follow the prompts

Method 2: Manual Installation
-----------------------------
1. Open Terminal
2. cd /Volumes/DELF*
3. Run: xattr -cr "DELF Practice.app"
4. Drag the app to /Applications
